#  Weather Checker App ☁

A simple yet powerful weather checking web app built using *HTML, CSS, and JavaScript*,  
that fetches *real-time weather updates* from an open public API.

This app allows users to quickly check weather details like temperature, humidity, and condition of any city worldwide.

---

##  Features
-  Search weather by city name.  
-  Displays temperature, humidity, and condition dynamically.  
-  Error handling for invalid city names.  
-  Fully responsive and modern interface.

---

## ⚙ Technologies Used
-  HTML  
-  CSS  
-  JavaScript  
-  OpenWeatherMap API
